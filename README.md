# Academia

Shared notes for easy access, for reference, when dispensing a contextually required responsibility, and when working with industries at various stages of their development.

These contains reference materials for topics studied as well as practice, and proof of concepts of certain topics, to serve as a quick guide for implementations.

The following are the broad topics under which the notes are maintained:

- [Business](/business/index.md)
- [Technology](/technology/index.md)

There are some highly interdisciplinary topics that will be focused on separately, which are:

- [Sustainability](./sustainability/index.md)
- [Decision Sciences](./decision_science/index.md)
- [Systems Theory](./systems/index.md)

If you'd like to contribute, you can do so, via the [GitHub repository](https://github.com/supratikchatterjee16/academia).
